## 🚀 HOW TO DEPLOY (20 minutes)

```
STEP 1: Read (5 min)
  └─ Open: QUICK_START_RENDER.md
  └─ Understand the 5 steps

STEP 2: Git Push (2 min)
  └─ git add .
  └─ git commit -m "Setup Render"
  └─ git push origin main

STEP 3: Create PostgreSQL (2 min)
  └─ Render Dashboard → + New
  └─ Select PostgreSQL
  └─ Name: arecom-db, Database: arecom_db
  └─ Copy the DATABASE_URL

STEP 4: Create Web Service (1 min)
  └─ Render Dashboard → + New
  └─ Select Web Service
  └─ Connect to GitHub: arecom-vet
  └─ Name: arecom-vet, Python 3.11

STEP 5: Add Environment Variables (3 min)
  └─ Web Service → Environment
  └─ Add all variables from .env.example
  └─ KEY ONES:
     - DEBUG=False
     - SECRET_KEY=[generated]
     - DATABASE_URL=[from PostgreSQL]
     - ALLOWED_HOSTS=arecom-vet.onrender.com

STEP 6: Deploy (5 min)
  └─ Wait for build (see logs)
  └─ Should say: "✅ Build completed successfully!"
  └─ App live at: https://arecom-vet.onrender.com

STEP 7: Verify (2 min)
  └─ Open: https://arecom-vet.onrender.com/admin
  └─ Login: admin / admin123
  └─ Change password immediately ✅

DONE! 🎉
```
